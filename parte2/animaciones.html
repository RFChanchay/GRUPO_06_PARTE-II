Crear un contenedor de animación

Todas las animaciones deben ser relativas a un elemento contenedor

Ejemplo

Ejemplo de JavaScript

Codigo 



<div id ="container">
  <div id ="animate">My animation will go here</div>
</div>

Dale estilo a los elementos

El elemento contenedor debe crearse con estilo = " position: relative".

El elemento de animación debe crearse con estilo = " position: absolute".

Codigo 



<!Doctype html>
<html>
<style>
#container {
width: 400px;
height: 400px;
position: relative;
background: yellow;
}
#animate {
width: 50px;
height: 50px;
position: absolute;
background: red;
}
</style>
<body>


<h2>Mi primera Animacion JavaScript</h2>


<div id="container">
<div id="animate"></div>
</div>


</body>
</html>

Código de animación

Las animaciones de JavaScript se realizan mediante la programación de cambios graduales en el estilo de un elemento.
Los cambios son llamados por un temporizador. Cuando el intervalo del temporizador es pequeño, la animación parece continua.
El código básico es:

Ejemplo

Codigo 



id = setInterval(frame, 5);

function frame() {
  if (/* test for finished */) {
    clearInterval(id);
  } else {
    /* code to change the element style */ 
  }
}

Crear la animación completa usando JavaScript

Ejemplo

Codigo 



<!DOCTYPE html>
<html>
<style>
#container {
width: 400px;
height: 400px;
position: relative;
background: yellow;
}
#animate {
width: 50px;
height: 50px;
position: absolute;
background-color: red;
}
</style>
<body>


<p><button onclick="myMove()">Click Me</button></p>


<div id ="container">
<div id ="animate"></div>
</div>


<script>
function myMove() {
let id = null;
const elem = document.getElementById("animate"); 
let pos = 0;
clearInterval(id);
id = setInterval(frame, 5);
function frame() {
if (pos == 350) {
clearInterval(id);
} else {
pos++; 
elem.style.top = pos + "px"; 
elem.style.left = pos + "px"; 
}
}
}
</script>


</body>
</html>
