El objeto HTMLCollection

El getElementsByTagName() método devuelve un HTMLCollectionobjeto.
Un HTMLCollectionobjeto es una lista similar a una matriz (colección) de elementos HTML.
El siguiente código selecciona todos los <p>elementos en un documento:

Ejemplo

const myCollection = document.getElementsByTagName("p");

Se puede acceder a los elementos de la colección mediante un número de índice.
Para acceder al segundo elemento <p> puedes escribir:

myCollection[1]

Codigo 


<!DOCTYPE html>


<html>

<body>

                    
<h2>Colecciones Ejemplo</h2>
                    
<p>Hola Mundo!</p>
                    
<p>Hola Ecuador!</p>
                    
<p id="demo1"></p>


<script>

  const myCollection = document.getElementsByTagName("p");
                    
  document.getElementById("demo1").innerHTML = "El HTML interno del segundo párrafo es: " + myCollection[1].innerHTML;
                    
</script>
               

</body>


</html>

Resultado

HTML HTMLCollection Longitud

La lengthpropiedad define el número de elementos en un HTMLCollection:

Ejemplo

myCollection.length

Codigo 


<!DOCTYPE html>

<html>

<body>
                    
<h2>Colecciones Ejemplo</h2>
                    
<p>Hola Mundo!</p>
                    
<p>Hola Ecuador!</p>
                    
<p id="demo2"></p>
                    
<script>

  const myCollection = document.getElementsByTagName("p");
                    
  document.getElementById("demo2").innerHTML = "El documento contiene " + myCollection.length + " parrafos.";
                    
</script>
                    
</body>

</html>

La lengthpropiedad es útil cuando desea recorrer los elementos de una colección:

Ejemplo

Cambia el color del texto de todos los elementos <p>:


Codigo 


<!DOCTYPE html>

<html>

<body>
                    
<h2>Colecciones Ejemplo</h2>
                    
<p>Hola Mundo!</p>
                    
<p>Hola Ecuador!</p>
                    
<p>Haga clic en el botón para cambiar el color de todos los elementos p.</p>
                    
<button onclick="myFunction4()">Intentalo!</button>
                    
<script>

  function myFunction4() {
  const myCollection4 = document.getElementsByTagName("p");
    for (let i = 0; i < myCollection4.length; i++) {
      myCollection4[i].style.color = "red";
      }
  }

  </script>
                    
</body>

</html>
